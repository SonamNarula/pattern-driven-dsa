## Patterns Guide

2. **Fast and Slow Pointers**: A method to identify cycles in sequences.
3. **Two Pointers**: Efficient approach for problems involving multiple collections.
4. **Backtracking**: Systematic way to explore all possible solutions by trying partial solutions.
